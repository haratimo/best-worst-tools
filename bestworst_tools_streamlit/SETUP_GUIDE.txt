HOW TO USE
==========

METHOD 1: Using the Launch Script (EASIEST)
--------------------------------------------

1. Install Streamlit (one-time only):
   pip3 install streamlit

2. Double-click run_app.sh
   OR in terminal:
   ./run_app.sh

3. Browser opens automatically!


METHOD 2: Manual Launch
------------------------

1. Install Streamlit:
   pip3 install streamlit

2. Open terminal in this directory

3. Run:
   streamlit run app.py

4. Open browser to http://localhost:8501

TYPICAL WORKFLOW
================

For Creating Trials:
--------------------
1. Prepare items.txt with your words/stimuli
2. Open the Streamlit app
3. Click "Create Trials"
4. Upload items.txt
5. Set N=8320, K=4 (or adjust as needed)
6. Click "Generate Trials"
7. Download the CSV
8. Upload to your experiment platform


For Scoring Data:
------------------
1. Collect participant data (CSV format)
2. Open the Streamlit app
3. Click "Score Data"
4. Upload participant files
5. Select scoring methods (e.g., Value, Best, Worst)
6. Click "Calculate Scores"
7. Download results
8. Analyze in R, Python, Excel, etc.

TROUBLESHOOTING
===============

Q: App won't start?
A: Make sure Streamlit is installed:
   pip3 install streamlit

Q: Import errors?
A: The app.py needs access to ../bestworst_tools_python3/scripts/
   Make sure that directory exists and has the Python scripts.

Q: Browser doesn't open?
A: Manually go to http://localhost:8501

Q: Port already in use?
A: Run with different port:
   streamlit run app.py --server.port 8502

Q: Can't upload files?
A: Check file format (.txt, .csv, or .tsv)
   Make sure file isn't empty

Q: Trial generation fails?
A: For "even" generators, N√óK must divide evenly by number of items
   Use suggested N values shown in the app
